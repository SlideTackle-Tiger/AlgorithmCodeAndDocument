# 动态规划
## 理论基础
动态规划包括哪种问题？
- 基础问题 - 如：斐波那契额数列、爬楼梯。
- 背包问题 - 如：装满背包的最大价值、装满背包的方案。
- 打家劫舍 - 如：抢劫房屋，但是不能连续抢。
- 股票问题 - 如：买卖股票的最大利润问题。
- 子序列问题 - 求子序列的各种情况回文、最长连续等。


动态规划类题目解题思路，主要用五部曲分析问题：
- DP数组定义以及下标的含义
- 递推公式
- dp数组如何初始化
- 遍历顺序
- 打印dp数组，分析上述四点是否与代码相同。
